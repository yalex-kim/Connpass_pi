🎫 JIRA 서버 관리

등록된 서버
URL 자동 판별 — atlassian.net 또는 atlassian.com 포함 시 Cloud (Basic Auth, email+API Token), 아니면 사내 Server (Bearer Token)
🔍 Gerrit 서버 관리
Gerrit 코드리뷰 서버를 등록합니다. Basic Auth 또는 Bearer Token을 지원합니다.
등록된 서버
인증 방식 — Basic Auth: Gerrit HTTP 자격증명 (Settings → HTTP Credentials) / Bearer: HTTP Bearer Token
